様々な国籍・年齢層の作業員が使用する製造現場で、誰でも迷わず使えるインターフェースを設計。シンプルで明確な画面設計により、スムーズなデジタル化を実現しました。

依頼背景

プロジェクトの依頼内容

製造現場の紙の作業日報をデジタル化し、リアルタイムでの状況把握を可能にするシステムのUIデザインを依頼されました。

特に重要な要件：
様々な国籍・年齢層の作業員が使用することを前提に、誰でも迷わず使えるインターフェースの設計が求められました。

現場の状況

製造現場では紙の作業日報を運用していましたが、記入ミスや集計の遅れにより実態把握が困難な状況でした。その結果、発生する不良に対して効果的な打ち手を打つことができず、品質改善が進まない状況が続いていました。

求められたこと：タイムリーに、手間をかけず、事実を把握できる仕組みの構築

解決すべき課題

🌍
現場作業員の多様性
デジタルリテラシーの差、様々な国籍・言語背景を持つ作業員への対応。日本語の読み書きレベルにも大きな差がある環境。
🏭
製造現場特有の環境
手袋着用必須、立ち作業中心、複数人が1台の端末を共有。素早い入力と切り替えが求められる厳しい環境。
⚡
異常時の迅速な対応
管理者の入力負担軽減（10画面→3画面）、作業停止時間の最小化、現場の焦燥感・不安感の軽減が必要。

Design Approach

1

要件分析と現場調査

実施内容
  • Webミーティングによる現場状況のヒアリング
  • 既存の紙日報の分析
  • 作業フローの可視化
得られたインサイト
  • 作業員は立ち作業が中心で、座って入力する時間が取れない
  • 様々な国籍の作業員が在籍（日本語の読み書きレベルに差がある）
  • 異常発生時は管理者への迅速な連絡が必須
  • 手袋を外さずに操作したいというニーズ
2

情報設計とワイヤーフレーム

画面遷移の最適化
課題：異常報告後、管理者が10画面を経由して入力する必要があり、作業再開が遅れる
解決：タイムライン表示を採用し、3画面に集約
効果：終わりが見える設計により、焦燥感を軽減し作業時間を短縮
主要な意思決定
  • QRコードによる簡易ログイン機能
  • かんばん読取機能とフリー入力の併用
  • タブ切り替えによる効率的な入力
3

ビジュアルデザインとコンポーネント設計

シンプルで明確なUI設計
  • 自分の行動と直結する画面遷移設計
  • ポップアップでやるべきことを明確化
  • アイコンと文字を併記し、言語の壁を超える工夫
アクセシビリティへの配慮
  • ボタンサイズを通常の1.5倍に拡大（手袋着用時の操作性向上）
  • 高コントラストカラーで視認性を確保
  • エラー表示を色だけでなくアイコンでも表現

Solution

「誰でも迷わない」シンプルで明確なインターフェース

自分の行動と直結する画面遷移、やるべきことが明確なポップアップ表示、 アイコンと文字の併記による直感的な理解。言語や文化の違いを超えて使えるユニバーサルデザインを実現しました。

📱

作業入力画面システム

大きなタッチターゲットとシンプルな画面構成で、誰でも迷わず操作可能。視覚的なフィードバックで確実な入力をサポート。

✓ タッチターゲット: 66×66px（通常の1.5倍）
✓ アイコンと文字の併記で言語の壁を解消
✓ ポップアップによる明確な行動誘導
🔐

ユーザー識別システム

QRコードによる簡単なユーザー切り替え。複数人での端末共有でも、誰が入力しているかを明確に表示。

✓ QRコードスキャンによる高速ログイン
✓ 視覚的にわかりやすいユーザー表示
✓ タイムアウト機能による自動ログアウト
⚠️

異常報告フロー（管理者向け）

3画面で完了する効率的な入力フロー（従来10画面）。タイムライン表示で進捗を可視化し、焦燥感を軽減。

✓ 画面数を70%削減（10画面→3画面）
✓ タイムライン表示で終わりが見える設計
✓ 必須項目の明確化で迅速な作業再開
📊

管理者向けダッシュボード

リアルタイムでの生産状況把握。異常発生時のアラート表示と、効率的な承認作業フロー。

✓ 生産実績のリアルタイム可視化
✓ 異常分類マスタの管理機能
✓ データエクスポート機能

Learning & Reflection

プロジェクトから得た学び

  • シンプルさの価値：機能を増やすことより、削ることの重要性を実感
  • 多様性への配慮：言語や文化の違いを前提とした設計の必要性
  • 現場の心理的側面：焦燥感や不安感といった感情面への配慮の重要性
  • 制約が創造性を生む：手袋着用などの制約が、より良いUIを生む原動力に

今後への活かし方

  • 多国籍・多様な背景を持つユーザーを前提としたデザインアプローチ
  • ポップアップやプログレス表示など、行動誘導を明確にする手法の活用
  • 業務の心理的負担を軽減する設計思想の継続
  • アイコンと文字の効果的な組み合わせによる情報伝達
